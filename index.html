I'm Aria Hassanpour, a Dentist,
researcher in AI, and React developer.

I graduated with a DDS degree from IAUTMS in 2023. Currently, I am immersed in Deep Learning research focused on medical image analysis. My work primarily involves utilizing advanced Deep Learning techniques, such as Convolutional Neural Networks (CNNs) and U-Net architectures, to analyze dental radiographs, with a special emphasis on Cone Beam Computed Tomography (CBCT) images. Additionally, I am enhancing my skills in React.js to develop web applications.

AI Projects

Lung Tumor Segmentation Capstone

View on GitHub

My solution for the Capstone Project of Udemy's Deep Learning with PyTorch for Medical Image Analysis course. The primary objective of this model is to detect and segment lung tumor on the CT scans of patients. Data was obtained from the Medical Segmentation Decathlon challenge, which consists of 63 CT scans of the Lung. A U-Net model was trained and evaluated with different metrics, including Dice and f1 scores.

React Projects

bookInsight

View on GitHub

a simple web application to find trending books and search for books in different categories. bookInsight is based on openlibrary API. This project is a personal project with the purpose of improving Javascript, React, and Chakra UI skills and learning to implement APIs. By default, the web page shows trending books and is set to view 60 results. The user has the option to change the color mode of the website to dark. For each book, you can see the cover, title, authors, availability of full text, and available languages. you can easily search for any keyword in any subject (category).
